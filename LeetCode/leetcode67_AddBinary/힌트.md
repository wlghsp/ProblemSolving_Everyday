# LeetCode 67. Add Binary - 힌트

## 문제 요약
두 개의 이진수 문자열을 **더한 결과**를 이진수 문자열로 반환하는 문제입니다.

### 예시
```
입력: a = "11", b = "1"
출력: "100"

설명:
  11  (3)
+  1  (1)
----
 100  (4)
```

```
입력: a = "1010", b = "1011"
출력: "10101"

설명:
 1010  (10)
+1011  (11)
-----
10101  (21)
```

### 제약 조건
- 1 ≤ a.length, b.length ≤ 10^4
- a와 b는 '0' 또는 '1'로만 구성
- 선행 0 없음 ("0" 제외)

---

## 💡 핵심 아이디어

**이진수 덧셈 = 십진수 덧셈과 같은 원리!**

```
십진수:  carry(올림)를 다음 자리로
이진수:  0+0=0, 0+1=1, 1+1=10(올림!), 1+1+1=11(올림!)
```

---

## 🔑 해결 전략

### 방법 1: 뒤에서부터 한 자리씩 더하기 ⭐

```python
def addBinary(self, a: str, b: str) -> str:
    result = []
    carry = 0
    i, j = len(a) - 1, len(b) - 1

    while i >= 0 or j >= 0 or carry:
        total = carry

        if i >= 0:
            total += int(a[i])
            i -= 1
        if j >= 0:
            total += int(b[j])
            j -= 1

        result.append(str(total % 2))   # 현재 자리
        carry = total // 2              # 올림

    return ''.join(reversed(result))
```

**동작 예시: "11" + "1"**
```
i=1, j=0: 1 + 1 = 2 → result='0', carry=1
i=0, j=-: 1 + 0 + carry(1) = 2 → result='00', carry=1
carry=1:  result='001', carry=0

reversed: "100"
```

### 방법 2: Python 내장 함수 활용

```python
def addBinary(self, a: str, b: str) -> str:
    return bin(int(a, 2) + int(b, 2))[2:]
```

- `int(a, 2)`: 이진 문자열 → 정수
- `bin(n)`: 정수 → 이진 문자열 ("0b..." 형태)
- `[2:]`: "0b" 접두사 제거

### 방법 3: 10진수 변환 (기존 풀이 방식)

```python
def addBinary(self, a: str, b: str) -> str:
    # 직접 2진→10진 변환
    def to_decimal(num):
        result = 0
        for i in range(len(num)):
            result += int(num[len(num) - i - 1]) * (2 ** i)
        return result

    # 직접 10진→2진 변환
    def to_binary(num):
        if num == 0:
            return "0"
        result = ""
        while num != 0:
            result += str(num % 2)
            num //= 2
        return "".join(reversed(result))

    return to_binary(to_decimal(a) + to_decimal(b))
```
