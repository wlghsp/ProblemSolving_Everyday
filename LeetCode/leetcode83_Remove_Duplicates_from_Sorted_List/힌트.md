# LeetCode 83. Remove Duplicates from Sorted List

## 문제 이해
- 정렬된 연결 리스트에서 중복된 값을 제거
- 각 값이 한 번씩만 나타나도록 함
- 정렬된 상태를 유지

## 핵심 아이디어
1. **정렬된 리스트의 특성**: 중복된 값들은 항상 연속해서 나타남
2. **현재 노드와 다음 노드 비교**: 값이 같으면 다음 노드를 건너뛰기
3. **In-place 수정**: 새 리스트를 만들지 않고 기존 리스트의 포인터만 수정

## 접근 방법

### 방법 1: 반복문 (Iterative)
```
현재 = head
while 현재 and 현재.next:
    if 현재.val == 현재.next.val:
        현재.next를 현재.next.next로 변경 (중복 노드 건너뛰기)
    else:
        현재를 다음으로 이동
```

### 방법 2: 재귀 (Recursive)
```
base case: head가 None이거나 head.next가 None이면 head 반환
recursive case:
    - head.next를 재귀 호출로 정리
    - head.val과 head.next.val이 같으면 head.next 반환
    - 다르면 head 반환
```

## 주의사항
- **빈 리스트**: head가 None인 경우 처리
- **단일 노드**: head.next가 None인 경우
- **모든 노드가 중복**: [1,1,1,1,1] → [1]
- **중복이 없는 경우**: [1,2,3] → [1,2,3]

## 시간/공간 복잡도
- **시간 복잡도**: O(n) - 리스트를 한 번 순회
- **공간 복잡도**: O(1) - 반복문 방식 / O(n) - 재귀 방식 (call stack)

## 예제 추적

### Example 1: [1,1,2]
```
초기: 1 → 1 → 2 → None
       ^
     current

1단계: 1.val == 1.val (다음 노드)
      1 → 2 → None
      ^
    current

2단계: 1.val != 2.val
      1 → 2 → None
          ^
        current

3단계: current.next가 None이므로 종료
결과: [1,2]
```

### Example 2: [1,1,2,3,3]
```
초기: 1 → 1 → 2 → 3 → 3 → None
       ^
     current

1 → 2 → 3 → 3 → None (첫 번째 1 제거)
^

1 → 2 → 3 → None (첫 번째 3 제거)
    ^

최종: [1,2,3]
```
