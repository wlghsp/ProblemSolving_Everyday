# 반복 풀이 진행표

## 복습 규칙
- **복습 주기**: 에빙하우스 망각곡선 기반 (1일 → 3일 → 7일)
- **졸업 조건**: 3회 연속 힌트 없이 풀이 성공
- **체크 방법**: 풀었으면 날짜 기입, 힌트 봤으면 (H) 표시

## 상태 설명
- `복습중`: 아직 졸업 조건 미달
