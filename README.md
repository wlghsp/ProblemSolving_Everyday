# ProblemSolving Everyday

매일 알고리즘 문제를 풀고, 반복 복습으로 체화하는 학습 프로젝트입니다.

## 학습 방식

1. **새 문제 풀이** - LeetCode/백준/프로그래머스 문제를 직접 풀기
2. **3단계 힌트** - 방향 힌트 → 구조 힌트 → 전체 풀이 순서로 단계적 학습
3. **반복 복습** - 에빙하우스 망각곡선 기반 (1일 → 3일 → 7일), 3회 연속 성공 시 졸업
4. **코드 리뷰** - 복잡도 분석, 엣지케이스, 개선점 피드백

## 폴더 구조

```
LeetCode/
├── leetcode{번호}_{문제이름}/
│   ├── Solution.py       # 풀이 코드 + 테스트 케이스
│   ├── Solution2.py      # 복습 풀이
│   └── 힌트.md            # 3단계 힌트 문서
│
└── 반복풀이/
    ├── progress.md       # 전체 진행표 (유형, 난이도, 복습 기록)
    └── week_YYYY_MM_DD.md # 주간 계획표
```
